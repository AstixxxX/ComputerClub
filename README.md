# YADRO_Project - ComputerClub

---

## 1. Build the Project

### Linux/macOS
```bash
chmod +x UnixBuilder.sh
./UnixBuilder.sh
```

### Windows
Double-click WindowsBuilder.bat or run in Command Prompt:
```cmd
WindowsBuilder.bat
```

---

## 2. Run the program

Execute the program with your log file:
```bash
./ComputerClub <log_file.txt>
```

### Example:
```bash
./ComputerClub ./example_log.txt
```
